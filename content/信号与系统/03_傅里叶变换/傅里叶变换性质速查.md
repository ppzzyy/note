# 傅里叶变换性质速查

> 所有傅里叶变换性质的快速参考

## CTFT 性质表

| 性质 | 时域 | 频域 |
|------|------|------|
| 线性 | ax₁(t)+bx₂(t) | aX₁(jω)+bX₂(jω) |
| 时移 | x(t-t₀) | X(jω)e^{-jωt₀} |
| 频移 | x(t)e^{jω₀t} | X(j(ω-ω₀)) |
| 时域微分 | x'(t) | jωX(jω) |
| 频域微分 | tx(t) | jdX(jω)/dω |
| 时域积分 | ∫x(τ)dτ | X(jω)/(jω)+πX(0)δ(ω) |
| 尺度变换 | x(at) | X(jω/a)/\|a\| |
| 卷积 | x(t)*h(t) | X(jω)·H(jω) |
| 乘积 | x(t)·y(t) | X(jω)*Y(jω)/(2π) |
| 共轭 | x*(t) | X*(-jω) |
| 翻转 | x(-t) | X(-jω) |

## 常用技巧

### 1. 对称性
- 实信号：X(-jω) = X*(jω)，幅度谱偶对称，相位谱奇对称
- 实偶信号：X(jω) 是实偶函数
- 实奇信号：X(jω) 是虚奇函数

### 2. 帕塞瓦尔定理
$$
\int_{-\infty}^{+\infty}|x(t)|^2 dt = \frac{1}{2\pi}\int_{-\infty}^{+\infty}|X(j\omega)|^2 d\omega
$$

### 3. 快速计算技巧
- 先求简单信号的变换
- 利用性质推导复杂信号

**例**：求 t·e^{-at}u(t) 的傅里叶变换
1. 已知 e^{-at}u(t) ↔ 1/(a+jω)
2. 利用频域微分：tx(t) ↔ jdX/dω
3. 结果：jd/dω[1/(a+jω)] = 1/(a+jω)²
