# 00_课程概述

## 课程核心

信号与系统解决的核心问题：**信号的表示 + 系统的分析**

### 信号的表示
- 时域表示：x(t) 或 x[n]
- 频域表示：X(jω) 或 X(e^jω)
- 复频域表示：X(s) 或 X(z)

### 系统的分析
- 时域：卷积 x(t) * h(t)
- 频域：乘法 X(jω)·H(jω)
- 稳定性、因果性判断

## 学习路线图

```
基础数学工具
    ↓
傅里叶级数（周期信号）
    ↓
傅里叶变换（非周期信号）
    ↓
  ┌─────────────┐
  ↓             ↓
拉普拉斯变换    离散傅里叶变换
(连续系统)      (离散信号)
  ↓             ↓
  └─────┬───────┘
        ↓
    Z变换
  (离散系统)
```

## 为什么需要这么多变换？

1. **傅里叶变换的局限**：要求信号绝对可积，很多信号不满足
2. **拉普拉斯变换**：引入衰减因子 e^(-σt)，使更多信号可变换
3. **Z变换**：离散版本的拉普拉斯变换

**核心思想**：引入额外变量（σ 或 r）来拓宽收敛范围

## 变换之间的关系

```
拉普拉斯变换 X(s), s = σ + jω
    ↓ 令 σ = 0
傅里叶变换 X(jω)
    
Z变换 X(z), z = r·e^(jω)
    ↓ 令 r = 1  
离散傅里叶变换 X(e^jω)
```
