# 00_电路基础概述

## 学科核心

电路分析的核心问题：**电压、电流与功率在电路中的分布规律**

### 基本物理量
- 电压 $U$（V）：电场力做功的能力
- 电流 $I$（A）：电荷的定向移动
- 功率 $P = UI$（W）：能量传递的速率
- 电阻 $R$（Ω）：对电流的阻碍程度

### 基本定律
- **欧姆定律**：$U = IR$
- **KCL**（基尔霍夫电流定律）：节点电流代数和为零
- **KVL**（基尔霍夫电压定律）：回路电压代数和为零

## 学习路线

```
基本元件与定律 → 等效变换 → 一般分析方法 → 电路定理
      ↓                                        ↓
 一阶/二阶动态电路 ← 运算放大器 ← 正弦稳态分析
```

## 知识体系

| 阶段 | 主题 | 核心内容 |
|------|------|----------|
| 1 | 基本元件与定律 | R/L/C 元件、KCL、KVL、功率 |
| 2 | 等效变换 | 串并联、Y-Δ 变换、电源等效 |
| 3 | 一般分析方法 | 节点电压法、网孔电流法 |
| 4 | 电路定理 | 叠加、戴维南、诺顿、最大功率传输 |
| 5 | 运算放大器 | 理想运放模型、反相/同相/差分放大 |
| 6 | 一阶动态电路 | RC/RL 零输入/零状态/全响应、三要素法 |
| 7 | 二阶动态电路 | RLC 串联/并联、过阻尼/临界/欠阻尼 |
| 8 | 正弦稳态分析 | 相量法、阻抗、交流功率、频率响应 |

## 与其他学科的联系

- **信号与系统**：电路是 LTI 系统的经典实例，传递函数 $H(s)$ 即电路的复频域分析
- **电磁学**：电路是麦克斯韦方程组在集总参数假设下的简化
- **FOC 电机控制**：电机等效电路、PWM 驱动、滤波器设计都依赖电路基础
