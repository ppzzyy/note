# RC电路的电流分析

## 核心问题

电容不是不允许电流通过吗？那 RC 电路怎么会有电流流过？

**答案是：电容不是完全不导电，而是只有在电压变化时才"导电"，稳定后就"断电"。**

核心公式：$i = C \dfrac{dv}{dt}$，只要 $dv/dt \neq 0$，就会有电流。

## 二阶 RC 阶梯电路分析

以两级 RC 级联电路为例：

```
U → R1 → e1 → R2 → e2
       |           |
      C1          C2
       |           |
      GND         GND
```

这其实是一个**二阶 RC 低通滤波器 / 状态变量模型**。

### 阶段一：刚接通电源（最关键）

电容电压不能突变 → 电容相当于短路：

- C1 ≈ 一根导线
- C2 ≈ 一根导线

电流路径：

```
U → R1 → e1 → C1 → 地
          → R2 → e2 → C2 → 地
```

✔ 有电流，而且还不小。

### 阶段二：充电过程中

- C1、C2 电压逐渐升高
- 电流逐渐减小
- 这是一个动态过程

### 阶段三：最终稳态（t → ∞）

- 电容电压不再变化 → $dv/dt = 0$ → $i = 0$
- 电容变成开路
- 整个电路最终没有电流
- e1、e2 变成某个固定电压

## 直觉比喻：能存电的桶

| 状态 | 类比 |
|------|------|
| 刚接电 | 桶是空的 → 水猛灌（大电流） |
| 充电中 | 水位上升 → 流速变慢 |
| 满了 | 不再流动（电流 = 0） |

## 常见误区

❌ "电容不允许电流通过"

✔ 正确说法：**电容不允许直流稳态电流通过，但允许变化电流（位移电流）**

## 控制系统视角

这个电路中的：

- C1 电压 → 状态变量 $v_1$
- C2 电压 → 状态变量 $v_2$

可以写成状态空间模型，是控制系统里"状态方程电路"的典型例子。
