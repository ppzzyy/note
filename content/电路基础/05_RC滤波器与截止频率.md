# RC 滤波器与截止频率

## RC 滤波的本质：阻抗的"拔河"

电容的复阻抗 $Z_C = \dfrac{1}{j\omega C}$ 随频率变化：

- 低频时 $Z_C$ 很大 → 相当于开路
- 高频时 $Z_C$ 很小 → 相当于短路

改变 R 和 C 的相对位置，得到两种基本滤波器：

| 类型 | 结构 | 效果 | 典型应用 |
|------|------|------|----------|
| 低通 (LPF) | R 在前，C 并联到地 | 阻断高频，保留低频 | 传感器信号平滑、抗混叠 |
| 高通 (HPF) | C 在前，R 并联到地 | 阻断直流（隔直），保留高频 | 提取信号边缘、音频去偏置 |

## 截止频率

截止频率 $f_c$ 是滤波器的性能分水岭。在此处输出功率降到输入功率的一半，即 **−3dB 点**。

当电阻阻抗等于电容阻抗模值时：$R = \dfrac{1}{2\pi f_c C}$

$$
f_c = \frac{1}{2\pi RC}
$$

物理意义：在 $f_c$ 处，输出信号相对于输入有 **45° 的相位滞后**（低通）或超前（高通）。

## 传递函数推导

### 从 KVL 出发

输入电压等于电阻压降与电容压降之和：

$$V_{in}(t) = i(t)R + V_{out}(t)$$

代入电容特性 $i(t) = C\dfrac{dV_{out}}{dt}$：

$$V_{in}(t) = RC\frac{dV_{out}}{dt} + V_{out}(t)$$

这是一个关于 $V_{out}$ 的一阶微分方程 → 系统中只有一个储能元件（电容），因此是**一阶系统**。

### 拉普拉斯变换求传递函数

对微分方程做拉普拉斯变换（$\dfrac{d}{dt} \to s$）：

$$V_{in}(s) = (sRC + 1)\,V_{out}(s)$$

$$
H(s) = \frac{V_{out}(s)}{V_{in}(s)} = \frac{1}{1 + sRC}
$$

令 $\tau = RC$（时间常数），则 $H(s) = \dfrac{1}{1 + s\tau}$。

### 复阻抗分压法（更直观）

s 域中：$Z_R = R$，$Z_C(s) = \dfrac{1}{sC}$

$$H(s) = \frac{Z_C}{R + Z_C} = \frac{\frac{1}{sC}}{R + \frac{1}{sC}} = \frac{1}{1 + sRC}$$

## 极点、截止频率与相位的关系

### 极点——频率响应的"锚点"

令分母为零：$1 + sRC = 0 \Rightarrow s_p = -\dfrac{1}{RC}$

极点位于 s 平面左半轴实轴上，到原点的距离即特征角频率：

$$\omega_c = |s_p| = \frac{1}{RC}$$

### 截止频率——能量的分水岭

由极点位置直接决定：

$$\omega_c = \frac{1}{RC} \quad (\text{rad/s}), \quad f_c = \frac{1}{2\pi RC} \quad (\text{Hz})$$

在截止频率处，增益下降 $1/\sqrt{2}$（即 −3dB）。

### 相位——滤波的"副作用"

$$\theta(\omega) = -\arctan(\omega RC) = -\arctan\left(\frac{\omega}{\omega_c}\right)$$

三个关键节点：

| 频率 | 相位 | 含义 |
|------|------|------|
| $\omega \ll \omega_c$ | $\theta \approx 0°$ | 信号几乎无延迟通过 |
| $\omega = \omega_c$ | $\theta = -45°$ | 单极点系统的恒定特征 |
| $\omega \gg \omega_c$ | $\theta \to -90°$ | 每个极点最终贡献 90° 滞后 |

### 联动总结

| 概念 | 表现 | 相互关系 |
|------|------|----------|
| 极点 $s_p$ | 决定系统的"骨架" | 极点离原点越近，截止频率越低 |
| 截止频率 $f_c$ | 决定系统的"带宽" | 此处幅度衰减 3dB，相位滞后 45° |
| 相位 $\theta$ | 决定系统的"实时性" | 频率越高滞后越重，截止频率是相位变化最快点 |

## 自然响应与特征角频率

### 自然响应（零输入响应）

无外部输入时，系统仅靠内部储能（如电容电荷）产生的反应。

$$V_{out}(t) = V_0 \cdot e^{-t/RC}$$

自然响应总是趋于衰减到零（稳定系统），衰减速度由 $\tau = RC$ 决定。

### 特征角频率

$$\omega_c = \frac{1}{\tau} = \frac{1}{RC}$$

它揭示两个转折点：

- **时域速度**：$\omega_c$ 越大 → $\tau$ 越小 → 系统达到稳态越快
- **频域边界**：即截止频率，Bode 图幅频曲线的转折点

### 两者联系

特征角频率 $\omega_c$ 就是自然响应中指数衰减的"速率因子"：衰减项 $e^{-\omega_c t}$。

| 维度 | 自然响应 | 特征角频率 |
|------|----------|------------|
| 视角 | 时域过程 | 频域/系统参数 |
| 决定因素 | 初始状态 + R,C | 仅由 R,C 决定 |
| 联系 | 衰减项含 $e^{-\omega_c t}$ | $\omega_c$ 越高衰减越快 |

## 工程应用：AF 系统中的 RC 滤波

传感器反馈信号（如 PDAF 像素值、CDAF 对比度评价）含高频噪声：

- **硬滤波**：ADC 采样前加 RC 低通，防止混叠
- **截止频率选择**：$f_c$ 太低 → 信号延迟 → AF 控制闭环相位裕度下降 → 对焦震荡
- **软硬结合**：用 EMA（指数移动平均）模拟 RC 滤波：$y[n] = \alpha \cdot x[n] + (1-\alpha) \cdot y[n-1]$

权衡：增加滤波让信号更平滑，但代价是响应变慢（极点限制带宽）且更容易震荡（相位滞后）。高级系统倾向用 MPC/Kalman 滤波在时域做状态估计，减少对硬件 RC 的依赖。

## Bode 图：截止频率前并非严格 0dB

理想渐近线模型中，截止频率前是 0dB 水平线，但实际曲线是**平滑过渡**的。

增益的精确表达式：

$$|H(j\omega)| = \frac{1}{\sqrt{1 + \left(\frac{\omega}{\omega_c}\right)^2}}$$

转换为分贝：

$$\text{Gain(dB)} = 20\log_{10}\frac{1}{\sqrt{1 + \left(\frac{\omega}{\omega_c}\right)^2}}$$

| 频率范围 | 增益 | 说明 |
|----------|------|------|
| $\omega \ll \omega_c$ | $\approx 0\text{dB}$ | 信号几乎无损通过 |
| $\omega = \omega_c$ | $-3\text{dB}$（约损失 29.3% 幅度） | 截止频率点 |
| $\omega \gg \omega_c$ | $-20\text{dB/dec}$ | 频率每增 10 倍，增益降 20dB |

关键细节：**幅度还未明显下降时，相位已经开始偏移**。

- 在 $0.1f_c$ 处：幅度损失仅约 0.04dB，但相位已滞后约 5.7°
- 在控制闭环中，这点相位滞后有时比幅值衰减更致命

工程启示：为保证信号完整性，通常将 $f_c$ 设为有用信号最高频率的 **2~5 倍**。

## LR 滤波器：为什么信号处理中少见

LR 滤波器截止频率推导（当电阻阻抗等于感抗时）：

$$|Z_L| = |Z_R| \Rightarrow 2\pi f_c L = R$$

$$f_c = \frac{R}{2\pi L}$$

注意方向差异：RC 中 R 越大 $f_c$ 越小（滤波越狠），LR 中 R 越大 $f_c$ 越大（滤波越轻）。

### 信号处理中不用 LR 的原因

| 问题 | 说明 |
|------|------|
| **体积** | 低频滤波需要巨大电感量，线圈体积比贴片电容大几个数量级 |
| **成本** | 电感比电容贵得多 |
| **EMI** | 电感本质是"小天线"，既发射也接收电磁干扰，在精密传感器电路中是噩梦 |
| **内阻 (DCR)** | 实际电感有直流电阻，改变电路增益；电容在直流下近乎完美开路 |

### LR 的不可替代场景

- **电源管理（DC-DC 开关电源）**：储能和滤除开关噪声，理想电感不消耗能量
- **射频/高频（GHz 级）**：所需电感值极小（nH 级），可用 PCB 走线实现
- **音响被动分频器**：大电感阻隔高频，只让低频进入低音单元

## LC 滤波器：更强的滤波，更大的麻烦

### 物理本质：一阶 vs. 二阶

| | RC | LC |
|---|---|---|
| 储能元件 | 1 个（电容） | 2 个（电感 + 电容） |
| 系统阶数 | 一阶 | 二阶 |
| 滚降斜率 | $-20\text{dB/dec}$ | $-40\text{dB/dec}$ |
| 截止频率 | $f_c = \dfrac{1}{2\pi RC}$ | $f_c = \dfrac{1}{2\pi\sqrt{LC}}$ |
| 稳定性 | 绝对稳定，无过冲 | 需小心设计阻尼，可能振荡 |

LC 滤波效果更强（更陡的滚降），但两个储能元件之间能量来回"踢球"，会产生**共振**。

### LC 在信号处理中的痛点

| 问题 | 说明 |
|------|------|
| **Q 值与振荡 (Ringing)** | 阻尼比太小时，截止频率附近出现增益峰值；时域阶跃响应有过冲和振荡 |
| **阻抗匹配敏感** | 性能高度依赖源阻抗和负载阻抗，负载变化则截止频率和效果跟着变 |
| **EMI** | 电感产生磁场，干扰周边精密电路 |
| **饱和电流** | 超过限制后感值骤降，滤波失效 |

### LC 的必选场景

- **电源滤波 (Power Integrity)**：大电流下电阻会烧毁，电感不耗能是唯一选择
- **射频/天线**：GHz 级别 RC 无法胜任，信号被电阻耗光
- **追求极致滚降**：需要隔离频率相近的信号时，$-40\text{dB/dec}$ 甚至更高阶组合

### 选型总结

> **管信号用 RC，管能量用 LR/LC。**
