# 什么是阻抗

## 一句话定义

**阻抗 = 交流世界里的"电阻"**，不仅限制电流大小，还影响相位。

$$Z = \frac{V}{I}$$

看起来和电阻一样，但 $Z$ 是**复数**，包含大小和相位。

## 为什么需要阻抗

直流电路里只有电阻，$V = IR$ 就够了。但交流电路中：

- 有电容、电感（不只是电阻）
- 电压和电流会"不同步"（有相位差）

单靠"电阻"不够用，所以引入"阻抗（Impedance）"。

## 阻抗的组成

$$Z = R + jX$$

| 部分 | 含义 | 特性 |
|------|------|------|
| $R$ | 电阻 | 真正耗能 |
| $X$ | 电抗 | 储能，不耗能 |
| $j$ | 虚数单位 | 表示 90° 相位偏移 |

## 三种元件的阻抗

| 元件 | 阻抗 | 频率特性 |
|------|------|----------|
| 电阻 | $Z_R = R$ | 不随频率变化 |
| 电容 | $Z_C = \frac{1}{j\omega C}$ | 频率越高，阻抗越小 |
| 电感 | $Z_L = j\omega L$ | 频率越高，阻抗越大 |

## 直觉比喻：水流模型

| 元件 | 类比 |
|------|------|
| 电阻 | 狭窄水管（一直阻碍） |
| 电容 | 弹性膜（只对变化敏感） |
| 电感 | 重的惯性水轮（不喜欢变化） |

## 阻抗的两个关键作用

1. **决定电流大小**：$I = V / Z$
2. **决定相位差**：
   - 电阻：电压和电流同步
   - 电容：电流**超前**电压
   - 电感：电流**滞后**电压

## 知识链路

```
阻抗 → RC/LC滤波器 → 频率响应 → Bode图
```

这条链打通后，看模拟电路会非常清晰。
