## 1. 三相无刷电机分类：BLDC vs PMSM

| 对比项 | BLDC（无刷直流电机） | PMSM（永磁同步电机） |
|--------|---------------------|---------------------|
| 反电动势波形 | 梯形波 | 正弦波 |
| 控制方式 | 六步换相（方波控制） | FOC矢量控制 |
| 转矩脉动 | 较大（换相时有冲击） | 极小（平滑） |
| 控制复杂度 | 低 | 高（需要FOC算法） |
| 典型应用 | 电动工具、风扇、无人机 | 机器人关节、伺服系统、电动汽车 |
| 噪音 | 较大 | 极低 |

**结论**：做FOC学习，目标是控制 **PMSM**（或高质量的BLDC用FOC控制），追求低噪音、高精度、低转矩脉动。

## 2. 电机基本结构与工作原理

### 2.1 核心部件

```
定子（Stator）：固定不动，绕有三相线圈（U/V/W相）
转子（Rotor）：旋转部分，内嵌永磁体（N/S极交替排列）
气隙（Air Gap）：定子与转子之间的间隙，磁场在此传递
```

### 2.2 工作原理（安培力驱动）

1. 向三相线圈通入电流 → 产生旋转磁场
2. 转子永磁体被旋转磁场"拖着走" → 转子旋转
3. **关键**：要让电机持续旋转，必须让定子磁场始终"超前"于转子磁场一个角度（通常90°电角度时力矩最大）

### 2.3 极对数（Pole Pairs）

- 极对数 `p` = 磁极对数（N极数量 = S极数量 = p）
- **电角度 = 机械角度 × p**
- 例：12极电机（p=6），转子转1圈（360°机械角），电角度转了 6×360° = 2160°
- 常见云台电机：7对极（p=7），航模电机：7~14对极

```
电角速度 ωe = p × ωm（机械角速度）
电角度   θe = p × θm（机械角度）
```

## 3. 反电动势（Back-EMF）

当电机旋转时，转子磁场切割定子线圈，产生感应电动势，方向与驱动电压相反，称为**反电动势（BEMF）**。

```
BEMF = Ke × ω
```
- `Ke`：反电动势系数（V·s/rad），电机固有参数
- `ω`：转子角速度（rad/s）

**重要意义**：
- BEMF限制了电机最高转速（当BEMF ≈ 供电电压时，电流趋近于零，无法再加速）
- 无感FOC（Sensorless FOC）就是通过检测BEMF来估算转子位置

## 4. 传感器方案对比

### 4.1 霍尔传感器（Hall Sensor）

```
原理：检测转子磁场方向，输出数字信号（0/1）
精度：低（每60°电角度才有一次跳变，共6个状态）
成本：极低（约5-10元）
适用：BLDC六步换相控制，不适合高精度FOC
```

**霍尔信号与换相关系**：
```
霍尔状态  H3 H2 H1  导通相
  1        0  0  1   +U -V
  2        0  1  0   +V -W
  3        0  1  1   +U -W
  4        1  0  0   +W -V
  5        1  0  1   +W -U
  6        1  1  0   +V -U
```

### 4.2 磁编码器 AS5600（推荐入门）

```
原理：检测轴端磁铁的旋转角度，输出12位绝对角度
精度：12位 = 4096分辨率/圈，约0.088°/LSB
接口：I2C（地址0x36）或模拟量输出
成本：约15-30元（含磁铁）
适用：SimpleFOC入门首选，安装简单
```

**AS5600接线**：
```
AS5600  →  MCU
VCC     →  3.3V
GND     →  GND
SDA     →  I2C_SDA（STM32: PB7 / ESP32: GPIO21）
SCL     →  I2C_SCL（STM32: PB6 / ESP32: GPIO22）
```

**注意**：磁铁需安装在电机轴正中心，与AS5600芯片距离约0.5-3mm，否则读数不准。

### 4.3 磁编码器 AS5048A（进阶）

```
精度：14位 = 16384分辨率/圈，约0.022°/LSB
接口：SPI（速度更快，适合高速电流环）
成本：约40-60元
适用：高性能FOC，ODrive等方案
```

### 4.4 无感方案（Sensorless）

```
原理：通过检测相电流和相电压，用观测器（如龙贝格观测器、扩展卡尔曼滤波）估算转子位置
优点：无需编码器，结构简单
缺点：低速性能差（BEMF太小），启动困难
适用：电动工具、风扇等不需要低速精确控制的场合
```

## 5. 电机关键参数

| 参数 | 符号 | 单位 | 说明 |
|------|------|------|------|
| 相电阻 | R | Ω | 定子线圈电阻，影响铜损 |
| 相电感 | L | mH | 定子线圈电感，影响电流响应速度 |
| 反电动势系数 | Ke | V·s/rad | 转速与BEMF的比例系数 |
| 力矩系数 | Kt | N·m/A | 电流与力矩的比例系数（Kt = Ke） |
| 极对数 | p | - | 影响电角度与机械角度的换算 |
| KV值 | KV | rpm/V | 每伏特对应的空载转速（KV = 1/Ke × 60/2π） |

**常用电机参数示例**：
```
5010航模电机（340KV）：
  KV = 340 rpm/V
  极对数 p = 7
  相电阻 R ≈ 0.1 Ω
  相电感 L ≈ 0.05 mH

GM6020云台电机：
  KV ≈ 13.7 rpm/V（低KV，大力矩）
  极对数 p = 7
  额定电压 24V
  额定力矩 1.2 N·m
```

## 6. 三相电流换相逻辑（六步换相）

六步换相是BLDC最基础的控制方式，理解它有助于理解FOC的优势：

```
步骤1: +U -V  (H1=1, H2=0, H3=0)
步骤2: +U -W  (H1=1, H2=1, H3=0)
步骤3: +V -W  (H1=0, H2=1, H3=0)
步骤4: +V -U  (H1=0, H2=1, H3=1)
步骤5: +W -U  (H1=0, H2=0, H3=1)
步骤6: +W -V  (H1=1, H2=0, H3=1)
```

**六步换相的缺点**（FOC要解决的问题）：
- 每步换相时，力矩有突变（转矩脉动）
- 电流不是正弦波，谐波大，效率低
- 无法精确控制力矩大小

**FOC的优势**：通过连续调节三相电流（正弦波），使合成磁场始终与转子磁场保持90°，实现最大力矩输出且无脉动。

## 8. 里程碑自测题

完成本任务后，你应该能够回答以下问题：

1. BLDC和PMSM的本质区别是什么？为什么FOC更适合PMSM？
2. 一台12极电机（p=6），转子转了30°机械角，对应多少电角度？
3. 为什么电机转速越高，可用力矩越小？（提示：BEMF）
4. AS5600和霍尔传感器各适合什么控制场景？
5. 六步换相为什么会产生转矩脉动？FOC如何解决这个问题？

**参考答案**：
1. BEMF波形不同（梯形vs正弦），控制方式不同；FOC需要正弦电流，与PMSM的正弦BEMF匹配，效率最高
2. 30° × 6 = 180° 电角度
3. BEMF随转速升高，可用电压差（Vbus - BEMF）减小，电流减小，力矩减小
4. AS5600适合FOC精确控制；霍尔传感器适合简单六步换相
5. 六步换相每60°才切换一次，切换瞬间电流突变导致力矩突变；FOC用连续正弦电流消除脉动
