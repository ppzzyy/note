# 任务5：3D打印结构件设计指南

> 阶段二 · 第5-8周 | 需求：2.2、6.3

---

## 概述

本指南帮助你利用家中的3D打印机，为FOC电机控制测试台架制作所需的结构件。主要包括三个部件：
1. **电机固定底座** — 固定无刷电机，防止测试时电机移位
2. **AS5600磁编码器安装座** — 精确对准电机轴中心，保证编码器读数准确
3. **整体测试台架** — 集成固定电机、驱动板、MCU的一体化平台

---

## 一、打印参数推荐

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 材料 | PLA 或 PETG | PLA易打印；PETG耐热性更好（驱动板发热场景） |
| 层高 | 0.2mm | 精度与速度的平衡点 |
| 填充率 | 40%~60% | 结构件需要一定强度 |
| 壁厚 | 3~4圈（1.2~1.6mm） | 保证安装孔周围强度 |
| 支撑 | 按需开启 | 悬空超过45°的结构需要支撑 |
| 打印速度 | 40~60mm/s | 精度优先，不要过快 |

---

## 二、部件1：电机固定底座

### 设计要点

- **安装孔位**：需根据所选电机底部安装孔距确定
  - 5010航模电机：底部4×M3螺丝，孔距约16mm×16mm
  - GM6020云台电机：底部4×M3螺丝，孔距约25mm×25mm
- **底座尺寸**：建议底板至少 80mm×80mm，厚度 5mm，保证稳定性
- **防滑设计**：底部可设计防滑纹路，或预留橡胶垫粘贴槽
- **走线槽**：侧面预留 3mm 宽走线槽，方便电机三相线引出

### 参考尺寸（以5010电机为例）

```
底板：80mm × 80mm × 5mm
电机安装台：直径50mm，高度3mm（略高于底板，防止电机底部螺丝触地）
安装孔：4× φ3.2mm通孔，孔距16mm×16mm，孔位居中
走线槽：宽4mm × 深3mm，从安装台侧面延伸到底板边缘
```

### Thingiverse 参考模型

- 搜索关键词：`5010 motor mount`、`BLDC motor base`、`SimpleFOC motor mount`
- 推荐链接：https://www.thingiverse.com/search?q=bldc+motor+mount

### FreeCAD 快速建模步骤

1. 新建文档 → Part Design → 新建草图（XY平面）
2. 绘制 80×80mm 矩形 → 拉伸 5mm（底板）
3. 在底板中心绘制 φ50mm 圆 → 拉伸 3mm（电机台）
4. 在电机台上绘制 4个 φ3.2mm 圆（孔距16mm）→ 打孔（贯穿）
5. 在侧面绘制走线槽草图 → 切除
6. 导出 STL → 切片打印

## 三、部件2：AS5600磁编码器安装座

### 设计要点（最关键！）

> ⚠️ **核心要求**：磁铁中心必须与电机轴中心同轴，偏心误差 < 0.5mm，否则编码器读数会出现周期性误差。

- **磁铁安装**：AS5600配套的径向磁铁（φ6mm×2.5mm）需粘贴在电机轴末端
  - 轴末端需加工或打印一个磁铁座（小圆柱帽），用502胶水固定磁铁
- **编码器PCB安装高度**：AS5600芯片面距磁铁表面 **0.5mm~3mm**（推荐1mm）
- **安装方式**：编码器座固定在电机后盖或底座上，不随轴旋转

### 参考尺寸

```
编码器座外径：与电机后盖外径匹配（5010约φ42mm）
编码器PCB安装面：中心开φ8mm孔（轴通过），4×M2螺丝固定PCB
安装高度调节：设计为可调节的卡扣或垫片结构
磁铁帽：φ7mm × 4mm 小圆柱，中心φ6mm凹槽深2.5mm（放磁铁）
```

### 安装校准方法

1. 打印磁铁帽，将磁铁压入凹槽，用502胶水固定
2. 将磁铁帽套在电机轴末端，确认同轴后用胶水固定
3. 安装编码器座，调整高度使AS5600芯片距磁铁约1mm
4. 上电后用I2C读取 `AGC` 寄存器（0x1A），值在 `0x70~0xB0` 之间表示磁场强度合适
5. 缓慢转动电机轴一圈，观察角度读数是否线性变化（无跳变）

## 四、部件3：整体测试台架

### 设计思路

将电机底座、驱动板安装位、MCU安装位集成在一块底板上，方便整体移动和调试。

### 推荐布局

```
┌─────────────────────────────────────┐
│  [电机+编码器]    [SimpleFOC Shield] │
│                                     │
│  [STM32/ESP32]   [电源接线端子]      │
│                                     │
│         走线槽（底部）               │
└─────────────────────────────────────┘
底板尺寸：200mm × 150mm × 4mm
```

### 各模块安装位设计

| 模块 | 固定方式 | 安装孔 |
|------|----------|--------|
| 电机底座 | M3螺丝 | 4×φ3.2mm，与底座对应 |
| SimpleFOC Shield | M3铜柱+螺丝 | 4×φ3.2mm，Arduino UNO孔距 |
| STM32/ESP32开发板 | M2.5铜柱+螺丝 | 4×φ2.7mm |
| 电源端子 | 卡扣或螺丝 | 按端子型号定 |

### 走线管理

- 底板背面设计 3mm 深走线槽，横纵各一条
- 电机三相线（粗）走独立槽道，与信号线分开，减少干扰
- 预留扎线孔（φ3mm），用于固定线束

---

## 五、打印后处理

1. **去支撑**：用尖嘴钳去除支撑材料，用锉刀打磨安装面
2. **安装孔处理**：用对应尺寸钻头（M3孔用φ3mm钻头）修整孔径，保证螺丝顺滑穿过
3. **螺纹孔**：如需螺纹，可用热熔铜嵌件（M3热熔螺母）压入，强度远高于直接攻丝
4. **表面处理**（可选）：用砂纸（400目→800目）打磨外表面，喷涂哑光漆
