## 四、编码器选型

### AS5600 磁编码器模块（推荐）

| 参数 | 规格 |
|------|------|
| 分辨率 | 12位（4096步/圈） |
| 接口 | I2C（地址0x36） |
| 工作电压 | 3.3V / 5V |
| 配套磁铁 | 需直径6mm径向充磁磁铁（通常随模块附赠） |
| 参考价格 | 8-15元/个 |
| 购买渠道 | 淘宝搜"AS5600 编码器模块" |

**安装要点**：
- 磁铁需固定在电机轴末端中心，与AS5600芯片距离 **0.5-3mm**
- 磁铁与芯片必须同轴，偏心会导致角度误差
- 建议用3D打印安装座保证对中精度（见任务5）

---

## 九、采购注意事项

1. **电机极对数确认**：收到电机后，用万用表测量三相绕组阻值，三相应基本相等（±5%以内），否则可能是次品
2. **驱动板兼容性**：国产SimpleFOC Shield兼容版质量参差不齐，建议选择评价好、有原理图的卖家
3. **AS5600磁铁**：必须是**径向充磁**（N/S极在圆柱侧面），不能用轴向充磁磁铁
4. **电源安全**：开关电源输出端建议加 **100μF/50V 电解电容** 滤波，防止电机制动时反向电压损坏驱动板
5. **备件建议**：AS5600模块建议买3-5个（便宜且容易损坏），驱动板建议备1块
