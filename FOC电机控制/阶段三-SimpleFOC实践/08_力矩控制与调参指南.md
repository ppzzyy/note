## 一、力矩控制原理

### 1.1 为什么力矩控制是FOC的核心

在FOC三环控制结构中，**电流环（力矩环）是最内层、最基础的环**：

```
位置指令 → [位置环PID] → 速度指令 → [速度环PID] → 力矩指令(Iq) → [电流环FOC] → PWM → 电机
```

力矩与电流的关系（PMSM）：

```
T = (3/2) × p × λ_f × Iq
```

- `T`：输出力矩（N·m）
- `p`：极对数
- `λ_f`：永磁体磁链（电机固有参数）
- `Iq`：q轴电流（直接控制量）

**结论：控制 Iq = 控制力矩**，Id 应保持为 0（最大力矩/电流比控制，MTPA）。

### 1.2 三种力矩控制模式对比

| 模式 | SimpleFOC枚举 | 原理 | 优点 | 缺点 |
|------|--------------|------|------|------|
| 电压力矩控制 | `TorqueType::VOLTAGE` | 直接设置Vq电压 | 无需电流传感器，简单 | 力矩精度低，受转速影响 |
| DC电流力矩控制 | `TorqueType::DC_CURRENT` | 控制总线电流 | 只需1个电流传感器 | 精度中等 |
| FOC电流力矩控制 | `TorqueType::FOC_CURRENT` | 直接控制Iq | 精度最高，真正FOC | 需要2个相电流传感器 |

## 二、硬件准备：电流传感器接入

### 2.1 SimpleFOC Shield v2 电流采样

SimpleFOC Shield v2 内置了基于分流电阻的相电流采样电路：

- **采样电阻**：0.01Ω（10mΩ）
- **运放增益**：约50倍
- **ADC引脚**（STM32F4）：A0（相A）、A1（相B）

```cpp
// 电流传感器初始化（内联采样，SimpleFOC Shield v2）
InlineCurrentSense current_sense = InlineCurrentSense(0.01f, 50.0f, A0, A1);
```

### 2.2 外置ACS712电流传感器（备选方案）

若使用无电流采样的驱动板，可外接ACS712：

```cpp
// ACS712-05B：灵敏度185mV/A，偏置2.5V
// 接A0、A1引脚
InlineCurrentSense current_sense = InlineCurrentSense(185e-3f, A0, A1);
```

## 三、完整代码示例

### 3.1 FOC电流力矩控制（最高精度）

```cpp
/**
 * FOC电流力矩控制示例
 * 硬件：STM32F4 + SimpleFOC Shield v2 + AS5600 + PMSM电机
 * 功能：通过串口实时调整目标力矩，体验柔顺控制
 */

#include <SimpleFOC.h>

// ===== 硬件配置 =====
// 无刷电机：极对数根据实际电机填写（5010电机通常为7对极）
BLDCMotor motor = BLDCMotor(7);

// 驱动器：SimpleFOC Shield v2 引脚定义
BLDCDriver3PWM driver = BLDCDriver3PWM(9, 5, 6, 8);  // PWM_A, PWM_B, PWM_C, EN

// 编码器：AS5600 磁编码器（I2C）
MagneticSensorI2C sensor = MagneticSensorI2C(AS5600_I2C);

// 电流传感器：内联采样
InlineCurrentSense current_sense = InlineCurrentSense(0.01f, 50.0f, A0, A1);

// 串口指令接口
Commander command = Commander(Serial);
void doTarget(char* cmd) { command.scalar(&motor.target, cmd); }
void doLimitCurrent(char* cmd) { command.scalar(&motor.current_limit, cmd); }

void setup() {
  Serial.begin(115200);
  SimpleFOCDebug::enable(&Serial);

  // 1. 初始化编码器
  sensor.init();
  motor.linkSensor(&sensor);

  // 2. 初始化驱动器
  driver.voltage_power_supply = 24.0f;  // 电源电压（V）
  driver.voltage_limit = 12.0f;         // 驱动电压限制（V）
  driver.init();
  motor.linkDriver(&driver);

  // 3. 初始化电流传感器
  current_sense.linkDriver(&driver);
  current_sense.init();
  motor.linkCurrentSense(&current_sense);

  // 4. 力矩控制模式配置
  motor.torque_controller = TorqueType::FOC_CURRENT;  // 真正的FOC电流控制
  motor.controller = MotionControlType::torque;        // 运动控制：力矩模式

  // 5. 电流环PID参数（最关键！）
  motor.PID_current_q.P = 5.0f;    // q轴电流环比例增益
  motor.PID_current_q.I = 1000.0f; // q轴电流环积分增益
  motor.PID_current_q.D = 0.0f;
  motor.PID_current_d.P = 5.0f;    // d轴电流环比例增益
  motor.PID_current_d.I = 1000.0f;
  motor.PID_current_d.D = 0.0f;

  // 6. 低通滤波器（抑制电流噪声）
  motor.LPF_current_q.Tf = 0.005f;  // 时间常数（s），越小响应越快但噪声越大
  motor.LPF_current_d.Tf = 0.005f;

  // 7. 安全限制
  motor.current_limit = 2.0f;   // 最大电流（A），根据电机额定值设置
  motor.voltage_limit = 12.0f;  // 电压限制（V）

  // 8. 初始化电机（自动校准编码器零点和电机方向）
  motor.init();
  motor.initFOC();  // 对齐编码器，会短暂转动！

  // 9. 注册串口指令
  command.add('T', doTarget, "目标力矩(A)");
  command.add('L', doLimitCurrent, "电流限制(A)");

  Serial.println("FOC力矩控制就绪！");
  Serial.println("发送 T0.5 设置0.5A目标电流（对应力矩）");
  Serial.println("用手拨动电机轴，感受柔顺控制效果");

  _delay(1000);
}

void loop() {
  // FOC核心循环（尽量不在此处添加耗时操作）
  motor.loopFOC();

  // 运动控制循环（可以比loopFOC慢）
  motor.move();

  // 处理串口指令
  command.run();
}
```

### 3.2 电压力矩控制（无需电流传感器，入门用）

## 四、电流环PID调参方法

### 4.1 调参目标

电流环是最内环，要求：
- **响应速度快**：带宽通常 1~5 kHz
- **无稳态误差**：积分项消除静差
- **无振荡**：避免电流震荡导致电机抖动

### 4.2 系统性调参步骤

```
步骤1：先将 I=0, D=0，只调 P
  → 从小到大增加P，直到电流响应出现轻微振荡
  → 记录此时P值为 Pu（临界增益）

步骤2：设置 P = 0.5 × Pu，开始加 I
  → 从小到大增加I，直到稳态误差消除
  → 典型值：I = 500~2000（取决于电机电感）

步骤3：验证阶跃响应
  → 通过SimpleFOC Studio观察Iq实际值跟踪目标值的波形
  → 目标：上升时间 < 1ms，超调 < 10%
```

### 4.3 典型参数参考值

| 电机类型 | P | I | LPF_Tf |
|---------|---|---|--------|
| 5010航模电机（低电感） | 3~8 | 500~1500 | 0.003~0.01 |
| GM6020云台电机（中电感） | 5~15 | 800~2000 | 0.005~0.02 |
| 步进电机改PMSM（高电感） | 10~30 | 200~800 | 0.01~0.05 |

### 4.4 使用SimpleFOC Studio实时调参

```
1. 打开SimpleFOC Studio（Python工具）
2. 连接串口，波特率115200
3. 在"Motor"标签页找到 PID_current_q 参数
4. 实时修改P/I值，观察波形变化
5. 找到最优参数后写入代码
```

## 五、柔顺控制实验

### 5.1 阻抗控制体验

力矩控制最直观的体验是**柔顺控制**（Compliance Control）：

```cpp
// 弹簧效果：电机像弹簧一样抵抗偏转
void loop() {
  motor.loopFOC();

  // 读取当前位置
  float current_angle = motor.shaft_angle;
  float target_angle = 0.0f;  // 目标位置（弹簧平衡点）

  // 弹簧刚度系数（N·m/rad）
  float spring_stiffness = 0.5f;

  // 计算力矩：偏离越远，力矩越大
  motor.target = -spring_stiffness * (current_angle - target_angle);

  motor.move();
  command.run();
}
```

### 5.2 阻尼效果

```cpp
// 阻尼效果：电机像在粘稠液体中转动
void loop() {
  motor.loopFOC();

  float velocity = motor.shaft_velocity;
  float damping = 0.1f;  // 阻尼系数（N·m·s/rad）

  // 力矩与速度反向，产生阻尼
  motor.target = -damping * velocity;

  motor.move();
}
```

### 5.3 重力补偿（单关节机械臂）

```cpp
// 单关节机械臂重力补偿
// 假设连杆质量m=0.1kg，长度L=0.1m
float m = 0.1f, L = 0.1f, g = 9.8f;

void loop() {
  motor.loopFOC();

  float angle = motor.shaft_angle;

  // 重力力矩 = m*g*L*cos(θ)
  float gravity_torque = m * g * L * cos(angle);

  // 补偿重力，使连杆在任意角度保持静止
  motor.target = gravity_torque;

  motor.move();
}
```

## 六、常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 电机剧烈抖动/振荡 | 电流环P增益过大 | 降低P值，增大LPF_Tf |
| 电机无响应或力矩很小 | 电流传感器未校准 | 重新运行 `current_sense.init()` |
| 电机发热严重 | 电流限制设置过高 | 降低 `motor.current_limit` |
| 编码器对齐失败 | initFOC()时电机被卡住 | 确保电机空载，重新上电 |
| 电流读数异常（负值/过大） | 电流传感器相序接反 | 交换A/B相电流传感器接线 |
