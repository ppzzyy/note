# 端到端自动驾驶技术应用到自动对焦

> 将自动驾驶领域的端到端学习技术应用到相机自动对焦系统

## 📋 项目背景

自动驾驶端到端技术（如 UniAD, VAD）在感知、预测、规划和控制方面取得了突破性进展。这些技术同样可以应用到相机自动对焦领域，实现更智能、更快速的对焦系统。

## 🎯 核心思路

```
自动驾驶端到端技术        →     自动对焦应用
├─ 视觉感知                    ├─ 场景理解辅助对焦
├─ 预测模型                    ├─ 预测性对焦
├─ 强化学习控制                ├─ 对焦策略优化
└─ 端到端优化                  └─ 端到端对焦
```
